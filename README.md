## Install


### sdcc

```
brew install sdcc
```

### stm8flash

- [stm8flash](https://github.com/vdudouyt/stm8flash)

```
git clone https://github.com/vdudouyt/stm8flash.git
cd stm8flash
make
make install
cd ..
rm -rf stm8flash
stm8flash
```
